# 无扩展名文件搜索 — 功能帮助

## 概述

"没有扩展名的文件"弹窗顶部的搜索栏支持类 Everything 语法的文件搜索，可快速从无扩展名文件列表中筛选目标文件。

搜索**不区分大小写**（除非使用 `case:` 函数），输入 `/` 和 `\` 等效（自动兼容 Windows 路径分隔符）。

---

## 1. 基础语法

### 1.1 关键词搜索

直接输入文本，在所有文件的**相对路径**中搜索。多个关键词以空格分隔时为 **AND** 关系（同时满足）。

| 示例 | 说明 |
|------|------|
| `py` | 路径中包含 "py" 的文件 |
| `src core` | 路径中同时包含 "src" 和 "core" 的文件 |

### 1.2 OR 逻辑

使用 `|` 分隔多个条件，满足任一即可。

| 示例 | 说明 |
|------|------|
| `readme \| dialogs` | 路径中包含 "readme" 或 "dialogs" |
| `src \| lib \| data` | 路径中包含 "src"、"lib" 或 "data" 中任意一个 |

### 1.3 NOT 逻辑

在条件前加 `!` 前缀，排除匹配该条件的文件。

| 示例 | 说明 |
|------|------|
| `!dll` | 排除路径中包含 "dll" 的文件 |
| `src !dialogs` | 路径含 "src" 但不含 "dialogs" |

### 1.4 精确短语

用双引号 `"..."` 包裹，匹配完整短语（含空格等特殊字符）。

| 示例 | 说明 |
|------|------|
| `"My Guide"` | 精确匹配 "My Guide"（含空格） |
| `"config.json"` | 精确匹配 "config.json" |

---

## 2. 通配符

| 通配符 | 含义 | 示例 | 匹配 |
|--------|------|------|------|
| `*` | 零个或多个字符 | `*.json` | 所有 .json 结尾的文件 |
| | | `src/*.py` | src 目录下所有 .py 文件 |
| | | `mylib*` | 以 "mylib" 开头的文件 |
| `?` | 单个字符 | `dialogs.p?` | dialogs.py、dialogs.pl 等 |

---

## 3. 函数语法

函数格式为 `函数名:参数`，用于精确条件过滤。

### 3.1 `name:` / `n:` — 文件名搜索

仅搜索文件的**名称部分**（不含目录路径），不会匹配路径中的目录名。支持通配符。

| 示例 | 说明 |
|------|------|
| `name:scanner` | 文件名含 "scanner" |
| `name:*.md` | 文件名以 .md 结尾 |
| `n:readme.*` | 文件名以 "readme." 开头（n 是简写） |

### 3.2 `path:` / `p:` — 路径搜索

在文件的完整相对路径中搜索（含目录）。可配合通配符使用。

| 示例 | 说明 |
|------|------|
| `path:src` | 路径含 "src" |
| `path:data` | 路径含 "data" |

### 3.3 `size:` / `s:` — 大小筛选

按文件字节大小过滤。支持比较运算符、范围和大小常量。

**比较运算符：**

| 语法 | 含义 |
|------|------|
| `size:>N` | 大于 N 字节 |
| `size:>=N` | 大于等于 N 字节 |
| `size:<N` | 小于 N 字节 |
| `size:<=N` | 小于等于 N 字节 |
| `size:=N` | 等于 N 字节 |
| `size:N` | 等于 N 字节（省略运算符） |

**大小单位：**

| 单位 | 字节数 |
|------|--------|
| `kb` | 1,024 |
| `mb` | 1,048,576 |
| `gb` | 1,073,741,824 |

支持小数：`1.5mb` = 1,572,864 字节。

**范围语法：**

| 语法 | 含义 |
|------|------|
| `size:min..max` | 在 min 到 max 之间（含边界） |
| `size:min-max` | 同上，`-` 等效 `..` |
| `size:..max` | 小于等于 max |
| `size:min..` | 大于等于 min |

**大小常量：**

| 常量 | 范围 |
|------|------|
| `empty` | 0 字节 |
| `tiny` | 1 B ~ 10 KB |
| `small` | 10 KB ~ 100 KB |
| `medium` | 100 KB ~ 1 MB |
| `large` | 1 MB ~ 16 MB |
| `huge` | 16 MB ~ 128 MB |
| `gigantic` | > 128 MB |

**示例：**

| 示例 | 说明 |
|------|------|
| `size:>1mb` | 大于 1 MB 的文件 |
| `size:10kb..1mb` | 10 KB ~ 1 MB 之间的文件 |
| `size:0..1000` | 不超过 1000 字节的文件 |
| `size:empty` | 空文件（0 字节） |
| `size:tiny` | 微小文件（1 B ~ 10 KB） |

### 3.4 `dm:` / `datemodified:` — 修改时间筛选

按文件修改时间过滤。时间格式为 ISO 8601：`YYYY-MM-DD` 或 `YYYY-MM-DDTHH:MM:SS`。

**比较运算符：**

| 语法 | 含义 |
|------|------|
| `dm:>date` | 晚于指定时间 |
| `dm:>=date` | 不早于指定时间 |
| `dm:<date` | 早于指定时间 |
| `dm:<=date` | 不晚于指定时间 |
| `dm:=date` | 精确匹配（含时分秒） |

**示例：**

| 示例 | 说明 |
|------|------|
| `dm:>2025-06-01` | 2025-06-01 之后修改的文件 |
| `dm:<2025-01-01` | 2025 年之前修改的文件 |
| `dm:>=2025-07-01T12:00:00` | 2025-07-01 12:00 之后修改的文件 |

### 3.5 `case:` — 大小写敏感

默认搜索不区分大小写。使用 `case:` 前缀可强制大小写敏感匹配。

| 示例 | 说明 |
|------|------|
| `case:README` | 仅匹配大写的 "README" |
| `case:*.TXT` | 仅匹配大写 .TXT 结尾的文件 |

### 3.6 `regex:` — 正则表达式

直接使用正则表达式进行匹配。

| 示例 | 说明 |
|------|------|
| `regex:.*\\.py$` | 匹配 .py 结尾的文件 |
| `regex:^src\\\\.*` | 匹配 src 目录下的文件 |
| `regex:config\\.json$` | 精确匹配 config.json 结尾 |

> **注意**：路径中使用 `\` 作为分隔符，正则中需用 `\\\\` 表示一个反斜杠。`/` 会被自动转换为 `\`，因此也可写作 `regex:^src/.*`。

---

## 4. 组合查询

以上语法可以任意组合使用，语法优先级为：

1. `"..."` 精确短语（最高）
2. `函数名:参数` 函数语法
3. `!` NOT 前缀
4. 空格 AND
5. `|` OR（最低）

**示例：**

| 查询 | 含义 |
|------|------|
| `src size:>1mb` | src 目录下且大于 1 MB |
| `!lib size:>1000` | 不在 lib 目录下且大于 1000 字节 |
| `"My Guide" \| readme` | 精确匹配 "My Guide" 或包含 "readme" |
| `src *.py !test` | src 目录下 .py 文件，排除 test 相关 |
| `path:data size:1kb..20kb` | data 目录下 1KB~20KB 的文件 |
| `dm:>2025-06-01 size:>1mb` | 2025-06-01 后修改且大于 1MB |
| `src \| lib size:>1mb` | （src 或 lib）且大于 1MB |

---

## 5. 操作说明

| 操作 | 方式 |
|------|------|
| 执行搜索 | 点击 **搜索** 按钮，或按 Enter 键 |
| 清除搜索 | 点击 **清除** 按钮，恢复显示全部文件 |
| 查看结果数 | 搜索完成后，右下角显示 `(匹配数/总数)` |

> **搜索始终在全部文件中进行**，不受勾选状态影响。搜索结果中已勾选的文件仍会置顶显示。

---

## 6. 注意事项

- 搜索仅在**无扩展名文件列表**中生效，不涉及带扩展名的文件。
- 搜索范围是文件的**相对路径**（相对于扫描根目录），而非仅文件名。
- 大小写不敏感匹配会同时匹配中文等 Unicode 字符。
- 使用 `*` 或 `?` 通配符时，其余字符会被自动转义，无需手动处理正则特殊字符。
- `regex:` 模式下，正则表达式直接传递给 Python `re` 模块，请确保语法正确。
